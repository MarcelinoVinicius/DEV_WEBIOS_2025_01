Em busca da Web (Uma historia de amor e pitufos)

Era uma vez em tão tão distante havia uma pitu... desculpe... Havia uma tribo de pitufos... que sonhavam em ser programadores vulgo Devs Web

Um belo dia anadando pelo bosque da cidade pitalandia, eles encontram uma cobra chamada python que para os intimos era chama de Py e ela ofereceu um futuro proibido chamando Java e o Pitufinho disse NÃO vou experimentar o CSS.

Mas logo ele mudou de ideia e acabou comendo o fruto proibido e acabou o conhecendo o capitão Java

Após ele conhecer o capitão Java ele dicidius programar por 40 dias e 40 noites, ele foi expulso do VsCode pelo HTML, quan ele achou que a tempestade havia acabado

veio um grande diluvio chamado Git e devastou todo o seu conhecimento pela parede e veio a chuva forte e o derrubou... Tragico FIM para pitufinho
